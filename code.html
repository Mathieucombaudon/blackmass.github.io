Tip 1

Use fetch() to get data from a server

Fetch is a new API that is built into the browser. It is a way to get data from a server. It is a replacement for XMLHttpRequest (XHR). It is a promise-based API. It is a lot easier to use than XHR.

Here is an example of how to use fetch() to get data from a server:


            const response = await fetch('https://jsonplaceholder.typicode.com/users')
            const data= await response.json()
            data.forEach(item => {
                let li = document.createElement('li')
                li.textContent = item.name
                list.appendChild(li)

            })

Tip 2

Use hightlight.js to display inline code to share

Highlight js is a combination of javascript and css to correctly and nicely display code in a web page. You just have to go to highlightjs CDN and add the files (JS + CSS) in the head of your HTML and write a little bit of script(you can put it in the head too)

Here is the function to add:


                hljs.highlightAll();
